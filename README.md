# Grey Note

**Receive anonymous messages from anyone. Minimalist. Private. Secure.**

Grey Note is a privacy-first anonymous messaging platform designed with a clean, text-heavy aesthetic. It allows users to share a personal link and receive messages without the sender needing to log in.

## 🛠️ Tech Stack

- **Frontend**: Vite + TypeScript + Vanilla CSS
- **Backend**: Supabase (Database & RLS)
- **Auth**: Clerk

## 🚀 Deployment

Optimized for **Vercel** with built-in SPA routing via `vercel.json`.
